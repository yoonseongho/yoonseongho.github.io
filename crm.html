CRM 시스템 구축

2025.11 ~ 2026.05

프로젝트 개요

고객 정보, 견적, 계약 이력을 통합 관리하기 위한 CRM(Customer Relationship Management) 시스템 구축 프로젝트입니다.

영업 담당자가 고객 및 프로젝트 정보를 보다 효율적으로 관리할 수 있도록 견적 등록, 견적 조회, 계약 조회 기능을 개발하였습니다.

Vue.js 기반 프론트엔드와 Spring Boot 기반 REST API를 활용하여 업무 시스템을 구축하였으며 사용자 편의성과 업무 효율성을 높이는 데 중점을 두었습니다.

기술 스택

Java Spring Boot Vue.js PostgreSQL REST API AG-Grid Git

담당 업무

  • 견적 등록 기능 개발
  • 견적 조회 기능 개발
  • 계약 조회 기능 개발
  • Spring Boot 기반 REST API 개발
  • Vue.js 화면 개발 및 API 연동
  • PostgreSQL 쿼리 작성 및 데이터 처리
  • 요구사항 분석 및 기능 개선 대응

주요 기능

견적 등록 기능

고객사 요청에 대한 견적 정보를 등록하고 관리할 수 있는 기능을 구현하였습니다.

  • 고객사 및 프로젝트 기반 견적 등록 기능 개발
  • 견적 품목별 금액 관리 기능 구현
  • 필수 입력값 검증 및 예외 처리 로직 적용
  • 견적 등록 및 수정 API 개발
  • 등록 데이터 이력 관리 기능 구현

견적 조회 기능

등록된 견적 정보를 다양한 조건으로 검색하고 관리할 수 있는 화면을 개발하였습니다.

  • AG-Grid 기반 견적 목록 화면 구현
  • 견적번호, 고객사명, 프로젝트명 조건 검색 기능 개발
  • 정렬 및 필터링 기능 구현
  • 상세 정보 조회 팝업 개발
  • 사용자 편의성을 고려한 검색 UI 구성

계약 조회 기능

계약 정보를 조회하고 관리할 수 있는 기능을 개발하였습니다.

  • 계약번호 및 고객사 기준 검색 기능 구현
  • 계약 상태별 조회 기능 개발
  • 계약 상세 정보 조회 화면 구현
  • 견적 및 프로젝트 데이터 연계 조회 기능 개발

기안문서함 기능

내부 결재 문서를 기안하고 결재 진행 현황을 확인할 수 있는 기능을 구현하였습니다.

  • 결재 문서 목록 조회 및 상세 보기 기능 개발
  • 결재양식, 상태, 문서번호 기반 검색 기능 구현
  • 결재 이력(결재선) 및 열람/참조자 표시 기능 구현
  • 휴가 신청 등 결재 양식별 결재내용 표시 처리

주요 화면

견적 등록

견적 목록과 상세 입력 폼을 좌우 분할 레이아웃으로 구성하였습니다. 견적 기간·타입·담당자 등 다양한 검색 조건을 제공하며, 첨부파일 업로드와 견적 비용 리스트 관리 기능을 함께 제공합니다.

견적 조회

AG-Grid 기반의 견적 목록 조회 화면입니다. 거래처구분·견적타입·담당자 조건 검색을 지원하며, 하단에 견적금액 리스트와 합계를 함께 표시합니다.

기안문서함

기안 문서 목록과 결재 상세 정보를 좌우 분할로 표시합니다. 결재 이력(결재선)과 열람/참조자 정보를 직관적으로 확인할 수 있으며, 결재내용(휴가 시작일·종료일·사유 등)을 양식별로 렌더링합니다.

계약 조회

계약 목록 그리드와 하단 상세 폼을 수직으로 구성하였습니다. 계약 기간, 거래처구분, 계약타입, 승인상태 조건 검색을 지원하며, 계약금액 리스트와 첨부파일 리스트를 하단 좌우에 나란히 배치하였습니다.

트러블 슈팅

대량 데이터 조회 성능 개선

견적 조회 화면에서 데이터가 증가함에 따라 조회 성능 저하 문제가 발생하였습니다.

문제 상황

  • 견적 데이터 증가로 목록 조회 시간이 지속적으로 증가
  • 여러 테이블 조인으로 인해 응답 속도 저하 발생
  • 사용자 검색 시 체감 성능 저하 발생

원인 분석

  • 실행계획(EXPLAIN) 분석
  • 검색 조건 컬럼 인덱스 미적용 확인
  • 불필요한 컬럼 조회 확인
  • 정렬 처리 시 Full Scan 발생 확인

개선 작업

  • 검색 조건 컬럼 인덱스 적용
  • 불필요한 JOIN 제거
  • 조회 컬럼 최소화
  • SQL 튜닝 수행
  • 조회 로직 최적화

결과

  • 목록 조회 속도 개선
  • 사용자 검색 편의성 향상
  • DB 부하 감소
  • 시스템 응답 속도 개선

프로젝트 성과

  • 견적 등록 및 조회 프로세스를 시스템화하여 업무 처리 효율 향상
  • 프로젝트 이력 통합 조회 기능 구축을 통해 과거 프로젝트 정보 활용성 향상
  • 계약 정보 조회 기능 구축으로 영업 및 운영 업무 지원
  • Vue.js + Spring Boot 기반 CRM 신규 구축 경험 확보
  • REST API 설계 및 개발 경험 습득
  • AG-Grid 기반 대용량 데이터 처리 경험 확보
  • PostgreSQL 쿼리 최적화 및 성능 개선 경험 확보
  • 요구사항 분석부터 개발, 테스트, 배포까지 전체 개발 프로세스 수행
